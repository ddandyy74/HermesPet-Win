# 社区行为准则 · Code of Conduct

## 我们的承诺

HermesPet 社区致力于为所有人提供一个友好、安全、包容的环境。

## 我们的标准

**鼓励的行为：**
- 友善和尊重地交流
- 接受建设性的批评
- 关注对社区最有利的事情
- 对其他社区成员表示同理心

**不可接受的行为：**
- 使用性别化的语言或图像
- 人身攻击、侮辱或贬低性评论
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息
- **声称自己是 HermesPet 的原作者或官方代表**
- **未经授权以 HermesPet 名义进行商业活动**

## 知识产权保护

本社区特别强调对知识产权的尊重：

1. **原创性**：HermesPet 由 Basion Wang 独立开发，任何声称拥有本项目原创权的行为将被视为严重违规
2. **归属要求**：使用本项目代码时必须遵守 Apache 2.0 许可证的归属条款
3. **品牌保护**：未经授权不得使用 HermesPet 名称和 Logo 进行推广

## 执行

违反行为准则的行为可以通过以下方式举报：
- 邮件：[basionwang@gmail.com](mailto:basionwang@gmail.com)
- GitHub Issue（标题带 `[CoC Violation]`）

所有举报将被认真审查并采取适当措施。

## 归属

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org/)，版本 2.1。
